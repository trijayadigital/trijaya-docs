# Pelaporan

> Panduan download rekapan transaksi di TriPay

TriPay menyediakan fitur **Download Rekapan Transaksi** melalui Dashboard untuk membantu Mitra dalam pemantauan, pembukuan, dan rekonsiliasi.

---

## Cara Mengakses Download Rekapan

1. **Login** ke Dashboard TriPay
2. Masuk ke menu **Transaksi** → **Download Rekapan**
3. Atur filter sesuai kebutuhan
4. Klik tombol **Generate**
5. **Download** file rekapan yang sudah di-generate

## Proses Generate Rekapan

### Langkah-langkah

1. **Pilih Filter**
  - Tentukan merchant (semua atau spesifik)
  - Tentukan status transaksi (semua atau spesifik)
  - Tentukan periode (hari ini, minggu ini, bulanan, atau custom)
2. **Generate Rekapan**
  - Klik tombol **Generate**
  - Sistem akan memproses dan membuat file rekapan
3. **Download Rekapan**
  - Setelah file siap, akan muncul di panel **Data Rekapan**
  - Klik ikon **download** untuk mengunduh file
  - File dapat di-refresh atau dihapus sesuai kebutuhan

## Rekonsiliasi

### Kewajiban

<callout color="amber" icon="i-lucide-alert-triangle">

Mitra wajib melakukan rekonsiliasi dan menyampaikan koreksi **paling lambat H+7 Hari Kalender** setelah tanggal transaksi.

</callout>

### Langkah Rekonsiliasi

1. Download Rekapan dari Dashboard
2. Bandingkan dengan data internal
3. Identifikasi selisih
4. Laporkan ke TriPay jika ada perbedaan

---

## Penyimpanan Bukti

<callout color="blue" icon="i-lucide-info">

Mitra wajib menyimpan bukti transaksi **minimal 12 bulan**.

</callout>
