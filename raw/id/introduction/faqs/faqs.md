# FAQs

> Pertanyaan yang Sering Diajukan tentang TriPay

Halaman ini berisi jawaban untuk pertanyaan-pertanyaan yang sering diajukan mengenai layanan TriPay.

## Umum

### Apa itu TriPay?

**TriPay** adalah layanan payment gateway yang membantu merchant menerima pembayaran online dari berbagai channel (Virtual Account, QRIS, E-Wallet, Retail) melalui satu integrasi.

### Apakah TriPay merupakan Bank atau E-Wallet?

**Tidak.** TriPay adalah **Operator Teknologi** yang bekerja sama dengan Penyedia Jasa Pembayaran (PJP) Rekanan untuk memfasilitasi transaksi. TriPay tidak menyimpan dana nasabah.

### Siapa yang bisa menggunakan TriPay?

- Pemilik bisnis online (e-commerce, marketplace)
- Developer yang membutuhkan payment gateway
- Penyedia layanan digital
- Semua pelaku usaha yang memerlukan pembayaran online

## Pendaftaran & Akun

### Bagaimana cara mendaftar di TriPay?

1. Kunjungi website TriPay
2. Klik **Daftar** atau **Register**
3. Isi data diri dan bisnis
4. Verifikasi email
5. Lengkapi KYC (upload KTP dan selfie)
6. Tunggu approval

### Berapa lama proses verifikasi akun?

Proses verifikasi biasanya memakan waktu **1-3 Hari Kerja** setelah dokumen lengkap disubmit.

## Merchant

### Bagaimana cara membuat Merchant baru?

1. Login ke Dashboard
2. Masuk ke menu **Merchant**
3. Klik **Tambah Merchant**
4. Isi informasi merchant
5. Pilih channel pembayaran
6. Submit untuk review

### Berapa lama approval Merchant?

Approval merchant biasanya **1-3 Hari Kerja**.

### Bisakah memiliki lebih dari satu Merchant?

**Ya.** Satu akun dapat memiliki beberapa merchant untuk bisnis yang berbeda.

### Apa syarat untuk mendapat settlement H+2?

Untuk Akun Bisnis:

- Minimum **30 transaksi sukses**
- Total transaksi minimum **Rp30.000.000**
- Akun minimal berusia **60 hari**

### Berapa biaya settlement?

**Rp7.500** per proses settlement.

### Apakah bisa withdrawal di hari Sabtu/Minggu?

Tersedia fitur **Weekend Withdrawal** dengan biaya tambahan. Cek Dashboard untuk ketersediaan.

## Callback / IPN

### Apa itu Callback/IPN?

**Callback (IPN - Instant Payment Notification)** adalah notifikasi otomatis yang dikirim TriPay ke server Anda saat status transaksi berubah (misal: pembayaran berhasil).

### Bagaimana cara mengatur Callback URL?

1. Login ke Dashboard
2. Masuk ke **Merchant** → merchant Anda
3. Isi **Callback URL** dengan URL endpoint Anda
4. Simpan

### Callback tidak diterima, kenapa?

Kemungkinan penyebab:

- URL tidak dapat diakses dari internet
- Firewall memblokir koneksi
- Server timeout (> 30 detik)
- SSL certificate tidak valid

## Integrasi

### Apakah ada dokumentasi API?

**Ya.** Dokumentasi API lengkap tersedia di Dashboard dan website TriPay.

### Platform apa saja yang didukung plugin TriPay?

- WooCommerce (WordPress)
- WHMCS
- Prestashop
- OpenCart
- Magento

### Bagaimana cara testing integrasi?

Gunakan **mode Sandbox** untuk testing tanpa transaksi nyata:

1. Dapatkan credential Sandbox di Dashboard
2. Gunakan URL API Sandbox
3. Simulasi pembayaran via Dashboard

## Troubleshooting

### Transaksi gagal, apa yang harus dilakukan?

1. Cek **status transaksi** di Dashboard
2. Cek **log callback** untuk error
3. Pastikan **saldo/limit** pelanggan mencukupi
4. Hubungi support jika masih bermasalah

### Dana tidak masuk sesuai jadwal settlement?

Kemungkinan penyebab:

- Settlement jatuh pada hari libur → diproses Hari Kerja berikutnya
- Rekening tujuan bermasalah
- Ada dispute atau hold pada dana

**Ya.** TriPay memiliki program referral yang memberikan komisi dari transaksi pengguna yang Anda referensikan.

### Bisnis apa saja yang tidak boleh menggunakan TriPay?

Berdasarkan PKS, TriPay **tidak melayani** bisnis yang terkait:

- Judi dan perjudian
- Narkotika dan obat terlarang
- Pornografi
- Money game / Ponzi scheme
- Kegiatan ilegal lainnya
