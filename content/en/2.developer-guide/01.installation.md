## Requirement
```
PHP <= 8.1
MySQL
Redis
Node 16.18
```

## Installation

1. Clone Repo
2. Copy `.env.example` ke `.env` caranya `cp .env.example .env`
3. Sesuaikan konfigurasi `.env`
4. Install dependensi Laravel `composer install`
5. Install dependensi JS npm i

## Running Development

1. Jalankan npm run watch-pool
2. Jalankan Laravel Built-In Server php artisan serve
3. Akses server
