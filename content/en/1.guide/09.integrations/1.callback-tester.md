**Callback Tester** adalah tool untuk menguji endpoint callback/webhook sebelum go-live.

---

## Pengertian

Callback Tester memungkinkan:
- **Mengirim test callback** ke URL endpoint
- **Memverifikasi** endpoint dapat menerima dan merespons
- **Debug** masalah callback tanpa transaksi nyata

---

## Cara Menggunakan

1. Login ke Dashboard TriPay
2. Menu **Developer** → **Callback Tester**
3. Masukkan **URL callback**
4. Pilih **tipe callback** (PAID, EXPIRED)
5. Klik **Kirim/Test**
6. Lihat hasil dan response

---

## Response yang Diharapkan

```json
{
  "success": true
}
```

---

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Connection refused | Pastikan server running |
| Timeout | Whitelist IP TriPay |
| SSL error | Gunakan SSL valid |
| 404 Not Found | Cek URL endpoint |
