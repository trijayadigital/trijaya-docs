**Mode Sandbox** adalah environment testing untuk integrasi tanpa transaksi nyata.

---

## Perbedaan Sandbox vs Production

| Aspek | Sandbox | Production |
|-------|---------|------------|
| **Transaksi** | Simulasi | Nyata |
| **Uang** | Tidak ada | Ada biaya |
| **Settlement** | Tidak ada | H+2/H+3 |
| **Credential** | Sandbox Key | Production Key |

---

## Mengakses Sandbox

1. Login ke Dashboard
2. Menu **Merchant** → pilih merchant
3. Tab **API** → copy credential Sandbox

---

## Cara Testing

1. **Buat transaksi** dengan API Sandbox
2. **Simulasi pembayaran** via Dashboard (Mark as Paid)
3. **Terima callback** di endpoint Anda

---

## Checklist Sebelum Production

::list{type="success"}
- [ ] Create transaction berhasil
- [ ] Callback diterima dengan benar
- [ ] Signature validation berfungsi
- [ ] Error handling terimplementasi
::
