TriPay menyediakan fitur **Download Rekapan Transaksi** melalui Dashboard untuk membantu Mitra dalam pemantauan, pembukuan, dan rekonsiliasi.

---

## Cara Mengakses Download Rekapan

1. **Login** ke Dashboard TriPay
2. Masuk ke menu **Transaksi** → **Download Rekapan**
3. Atur filter sesuai kebutuhan
4. Klik tombol **Generate**
5. **Download** file rekapan yang sudah di-generate

---

## Filter Rekapan

### 1. Merchant

Pilih merchant yang ingin direkap:

| Opsi | Keterangan |
|------|------------|
| **Semua Merchant** | Rekapan untuk semua merchant yang dimiliki |
| **Merchant Spesifik** | Rekapan untuk satu merchant tertentu |

### 2. Status Transaksi

Pilih status transaksi yang ingin direkap:

| Opsi | Keterangan |
|------|------------|
| **Semua Status** | Semua status transaksi (UNPAID, PAID, EXPIRED, dll) |
| **Status Spesifik** | Pilih status tertentu (misal: hanya PAID) |

### 3. Periode

Pilih periode waktu untuk rekapan:

| Opsi | Keterangan |
|------|------------|
| **Hari Ini** | Transaksi pada hari ini saja |
| **Minggu Ini** | Transaksi dalam 7 hari terakhir |
| **Bulanan** | Transaksi dalam bulan berjalan |
| **Custom** | Pilih rentang tanggal secara manual |

---

## Proses Generate Rekapan

### Langkah-langkah

1. **Pilih Filter**
   - Tentukan merchant (semua atau spesifik)
   - Tentukan status transaksi (semua atau spesifik)
   - Tentukan periode (hari ini, minggu ini, bulanan, atau custom)

2. **Generate Rekapan**
   - Klik tombol **Generate**
   - Sistem akan memproses dan membuat file rekapan

3. **Download Rekapan**
   - Setelah file siap, akan muncul di panel **Data Rekapan**
   - Klik ikon **download** untuk mengunduh file
   - File dapat di-refresh atau dihapus sesuai kebutuhan

---

## Data Rekapan

Setelah di-generate, file rekapan akan menampilkan:

| Informasi | Keterangan |
|-----------|------------|
| **Nama File** | Nama file rekapan dengan format tanggal |
| **Ukuran File** | Ukuran file dalam KB/MB |
| **Aksi** | Refresh, Download, atau Hapus file |

---

## Format File

File rekapan tersedia dalam format yang dapat diunduh untuk keperluan analisis dan pembukuan.

---

## Rekonsiliasi

### Kewajiban

::callout{icon="i-lucide-alert-triangle" color="amber"}
Mitra wajib melakukan rekonsiliasi dan menyampaikan koreksi **paling lambat H+7 Hari Kalender** setelah tanggal transaksi.
::

### Langkah Rekonsiliasi

1. Download Rekapan dari Dashboard
2. Bandingkan dengan data internal
3. Identifikasi selisih
4. Laporkan ke TriPay jika ada perbedaan

---

## Penyimpanan Bukti

::callout{icon="i-lucide-info" color="blue"}
Mitra wajib menyimpan bukti transaksi **minimal 12 bulan**.
::
